What is SpaceMining?

SpaceMining is a single-agent reinforcement learning environment simulating asteroid mining in space. The agent must collect resources from asteroids, deliver them to the mothership, manage energy, and avoid obstacles.

Features

  • Physics-based 3D space mining simulation
  • Resource & energy management
  • Partial observability and dynamic environment
  • Customizable reward structure and scenario

Task Description

The agent (mining robot) is deployed in a 3D space with randomly distributed asteroids and a central mothership. The agent must:

  • Search for and mine resource-rich asteroids
  • Efficiently manage energy and return to the mothership for recharging
  • Avoid collisions with obstacles
  • Deliver resources to maximize efficiency

Training Configuration

The environment supports PPO and other RL algorithms. See space_mining_ollama.yml for full configuration and hyperparameters. 

max_episode_steps: 1600
grid_size: 100
max_asteroids: 25
max_resource_per_asteroid: 80
observation_radius: 50
policy: MlpPolicy
learning_rate: 0.0003
total_timesteps: 3,000,000
